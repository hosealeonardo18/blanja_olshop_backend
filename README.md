
# Tugas Basic Backend

## Third Party Module



| Third Party       |  NPM Install                |
| :--------         | :------------------------- |
| Express           | `npm i express` |
| Nodemon           | `npm i nodemon --D` |
| Morgan            | `npm i morgan` |
| PostgreSQL        | `npm i pg` |
| Dotenv            | `npm i dotenv` |
| CORS              | `npm i cors` |
| Eslint            | `npm i eslint` |
| Http-errors     | `npm i http-errors` |
